Projects

Key examples of some projects I've done in my last positions:

AWS Reserved Instances Recommendator - Based on AWS obscure billing files, a tool to analyze running workloads and create a purchase recommendation based on economic, coverage and usage risk factors.
Microservices platform + DevOps pipelines - Build from zero a new multi-site platform based on Kubernetes and Docker plus all the relevant O&M toolset and procedures to enable CI/CD.
Real-Time service Cloud migration - Design and perform a cloud migration of one the product components of a media (VoIP) platform, including the cloud adoption roadmap and performance testing of real-time ssystems
Hybrid Cloud Management: Infrastructure as Code (IaC) - Design and implement the IaC of an hybrid-cloud infrastructure (AWS + VMWare) using Terraform, Packer and Ansible
Multibranching (Blue/Green) - Design on how to run multiple versions of same component in parallel and how to do smooth traffic transition betwen branches to route customers traffic to especific versions.
Measure Everything - Devise a new monitoring philosophy based on metrics collection and anomalies detection.
E2E Probes - Develop custom E2E probes based on Raspberry Pi (embeded linux) to worldwide monitor the ‘real user experience’ in black box monitoring mode.
